# FHH-Fullstack

## Tech Stack
- Frontend: Nuxt 3, TailwindCSS
- Backend: FastAPI
- Database: SQLite

## Features
- Search diagnosis codes
- Create consultation
- View consultation list

## How to Run

### Backend
uvicorn main:app --reload

### Frontend
npm install
npm run dev

## API Endpoints
- GET /diagnosis
- POST /consultation
- GET /consultation
